# Firmware

This folder contains the ESP32 firmware for the Soft Tactile Force Sensor.

## Hardware

- ESP32 DevKit V1
- MLX90393 3-axis magnetometer

## Framework

- PlatformIO
- Arduino Framework

## Libraries

- Adafruit MLX90393 Library

## Build

Open the `firmware` folder in VS Code with PlatformIO and click **Build**.

## Upload

Connect the ESP32 and click **Upload** in PlatformIO.
